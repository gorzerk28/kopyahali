# Ezan ses dosyası yerleşimi

İlk etap yerel (aynı domain) kullanım için ezan sesini bu klasöre koyun:

- Dosya adı: `assets/ezan.mp3`
- Öneri: mümkünse 128kbps MP3, UTF-8 olmayan özel karakter içermeyen dosya adı

Uygulama artık varsayılan olarak bu yolu deneyecek:

- `/assets/ezan.mp3`

Eğer dosya bulunamazsa veya tarayıcı kaynağı açamazsa uygulama otomatik olarak
yedek varsayılan ezan kaynağına düşer.
